
# Task Manager Python

A simple task manager built in Python that allows users to create, mark as completed, and delete tasks. The data is stored locally in a JSON file, making it easy to persist tasks across sessions.

## Installation

1. Clone this repository to your local machine:

```bash
git clone https://github.com/Dxrki44/task-manager-python.git
```

2. Navigate to the project directory:

```bash
cd task-manager-python
```

3. Run the program:

```bash
python task_manager.py
```

## Usage

Once the program is running, you can choose from the following options in the menu:

1. **View tasks** – Display all current tasks.
2. **Add a task** – Add a new task.
3. **Mark a task as done** – Mark a task as completed.
4. **Delete a task** – Remove a task from the list.
5. **Quit** – Exit the program.

## Features

- Add new tasks
- Mark tasks as done
- Delete tasks
- View all tasks
